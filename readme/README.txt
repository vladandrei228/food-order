This is the installation guide as this project have been made on localhost:
1) Install Xampp - https://www.apachefriends.org/ro/index.html
2) Open Xammp Control Panel and start Apache and MySQL
3) Go to: http://localhost/phpmyadmin/
4) Create a new database called food-order
5) Import the sql file from this folder
6) Move food-order folder to the Xampp folder into htdocs folder
7) Use the following links in browser:
	-localhost/food-order for Customer side
	-localhost/food-order/admin for Admin side

Enjoy!
